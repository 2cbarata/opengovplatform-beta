Data Tool
=========

Supported reply format (``format=`` parameter) types:

* json
* jsonp (default)
* csv
* xml

API Documentation
+++++++++++++++++
   
Use::

> `HOST?url=...`

Parameters
==========

`url  => URL of data resource, such as XLS or CSV file. This is required  `

`format  => Output format. Currently json, csv,jsonp,xml are supported `

**XLS Parameters:**

`worksheet => Worksheet number`
