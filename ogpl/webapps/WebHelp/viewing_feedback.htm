Viewing Feedback

The All Feedbacks page displays the list of feedback received from users of the OGPL Web site. The most recent feedback appears at the top of the list. You can view details of each feedback, such as the action status, category, source, and current status of a feedback.

To view feedback:

  1. In the navigation menu, click the Feedback link.
    The Assigned Feedbacks page appears.
    Note: when you click the Feedback link, the My List view appears by default.

  2. Click the appropriate tab. The following table lists each tab and its description.

Table: Description of the tabs for viewing feedback

Tab

Description

All

Displays all the feedback from various pages of the OGPL Web site

My List

Displays the feedback assigned to the URM Administrator.

Contact Us

Displays  the feedback from the Contact Us page

Feedback Through Website

Displays the feedback from the Feedback page

Ratings

Displays the feedback from the Ratings page

Suggest Datasets

Displays the feedback from the Suggest Dataset page
