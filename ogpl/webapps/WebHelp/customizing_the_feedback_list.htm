Customizing the Feedback List

You can add or remove columns from the page that displays the list of feedback. You can also specify the location where you want the column to appear on the page.

To customize the page that displays the feedback:

  1. On the navigation menu, click the Feedback link.
    The Assigned Feedbacks page appears.

  2. Click the Manage Fields tab.
    The Feedback list columns page appears.

  3. Select the check box for each column that you want to display on the feedback list.

  4. The position number indicates the location where a column must appear on the feedback list.
    In the Position box, enter the number that corresponds to the location where this column must appear. For example, if you want Action status to appear as the first column, enter 1 in the Position box.

  5. Click Submit.
    The columns that you selected appear on the feedback list according to the position that you specified.

Note: If you do not specify the position for a column, that column appears as the first column on the feedback list. If you specify the same position for multiple columns, the columns appear in alphabetical order on the feedback list.

 

Example:If you set the position for Action Status and Feedback ID to 1, Action Status appears as the first column and Feedback ID appears as the second column on the feedback list.
