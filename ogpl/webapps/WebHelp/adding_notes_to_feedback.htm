Adding Notes to Feedback

You can add notes to provide comments or additional details for a feedback, when the feedback is in any of the following states:

  • Assigned

  • Replied

  • Reviewed

  • Forwarded

  • Closed

  • Archived

 

To add a note to a feedback:

  1. On the navigation menu, click the VRM link.
    The VRM page appears.

  2. Click the tab that contains the feedback to which you want to add a note.

  3. Optional: You can filter the feedback list. For more information, see Filtering the List of Feedback.

  4. In the row of the feedback for which you want to add a note, click the Edit this record icon .
    Note: If the Edit this record icon  is not available for a feedback, click the Preview this record icon .
    The Feedback Details Edit page appears.

  5. Click the Notes tab.
    The Feedback Details page appears.

  6. In the Add note field, enter your notes and comments for this feedback.
    Note: You can enter up to 3000 characters.

  7. Click Submit.
    The note is added to the feedback.
