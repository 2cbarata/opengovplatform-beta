Discarding a Feedback

You can discard an inappropriate feedback sent by a user of the OGPL Web site. When you discard a feedback, the feedback is not deleted from the database, only the status of feedback changes.

To discard a feedback:

  1. In the navigation menu, click the Feedback link.
    The     Assigned Feedbacks page appears.

  2. Click the All tab.
    The All Feedbacks page appears.

  3. Click the View link for the feedback that you want to discard.
    The     Feedback Details page appears.

  4. Click Discard.
    The feedback is discarded.

Tip: If you want to reuse a discarded feedback, change the status of the discarded feedback to Reinitiate.
